# ARA-Graph-Assistant

Autonomous Research Assistant with Knowledge Graph
What It Does Imagine you ask it : "Research the impact of artificial intelligence on healthcare."

The system:
Searches multiple sources - academic papers, articles, reports, trusted websites. <br/>
Reads and understands content - extracts key concepts, facts, relationships<br/>
Builds a knowledge graph - creates nodes for concepts (AI, healthcare, diagnosis, treatment) and edges showing how they relate <br/>
Identifies insights - finds where sources agree, disagree, or contradict each other<br/>
Generates a comprehensive report - structured summary with proper citations<br/>
Lets you explore interactively - visual graph where you click on concepts to dive deeper, see connections, trace back to original sources<br/>
Answers follow-up questions - "How is AI used in radiology?" using the knowledge it already gathered<br/>
